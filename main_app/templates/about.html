Stages of Development:

  • Ideation: The project is in the brainstorming phase, exploring concepts and scope.
  • Planned: The project has a defined plan and schedule but hasn't started yet.
  • In Progress: Work on the project is actively ongoing.
  • On Hold: The project is paused temporarily, but not abandoned.
  • Refactoring: Work is focused on improving code or structure without changing functionality.
  • Testing: The project is being tested for issues, bugs, or performance concerns.
  • Deployed: The project is live and accessible in production.
  • Completed: All work is done, and the project has been delivered successfully.
  • Archived: The project is no longer actively maintained but still accessible for reference.
  • Cancelled: The project was terminated without completion.
